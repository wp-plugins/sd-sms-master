Connects to an SD SMS Master and can install and configure slaves and phones, create and monitor orders.

== Description ==

This plugin is a part of the SD SMS Master package. It connects to an SD SMS Master and can:

* populate the database and create a new admin user
* view the status of the master
* add / remove / edit orders
* add / remove / edit phones
* add / remove / edit slaves
* add / remove / edit users

Full documentation can be found by downloading the lib part from the plugin site and generating the API docs.

== Installation ==

1. Unzip and copy the zip contents (including directory) into the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress.
